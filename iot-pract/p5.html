1. Python script to write RFID (Write.py)

• Save the above script as Write.py .
• Click on Build tab and compile the program.
• After successful compilation, again click on Build and execute the program.
• Enter a name for your tag (eg. New Data: RFID Demo).
• Press enter (jot down what you have written.).
• Place the tag near the RFID reader wait for message written.
• Your RFID is written successfully with New Data.
• Now create a new python file and type the code to read RFID.

2. Python script to read RFID (Read.py)

• Save the above script as Read.py
• Click on Build tab and compile the program.
• After successful compilation, again click on Build and execute the program.
• Place the tag near the RFID reader wait for the RFID to read it.


Software Setup:

• Open Terminal and type the following commands
➢ sudo apt-get update
➢ sudo apt-get upgrade

• Create a new folder named 4digitTime using following command
➢ mkdir 4digitTme

• Open the folder using following command
➢ cd 4digitTme

• To see the complete path of the folder type following command
➢ pwd

• Download the python script tm1637.py using following command under 4digitTime folder
➢ wget https:raspberrytips.nl/files/tm1637.py

• You can check the list of available files in your folder using following command
➢ ls
