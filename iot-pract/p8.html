Practical No : 8


Aim : Interfacing camera module with Raspberry Pi.

A. For capturing image:
B. For capturing video:

Software Setup:

• Open Terminal and type the following commands
➢ sudo apt-get update
➢ sudo apt-get upgrade

• Download and install Pi camera library:
➢ sudo apt-get install python-pip3
➢ pip3 install picamera

• Enable Camera functionality on Raspberry Pi
➢ sudo raspi-config
➢ select Interfacing options
➢ select the camera option
➢ enable

• Reboot Raspberry Pi.
➢ sudo reboot
