Practical - 4


Aim : Setting up wireless access point using raspberry Pi.



Software Setup:

• Open Terminal and type the following commands
➢ sudo apt-get update
➢ sudo apt-get upgrade

• Install RaspAP by typing the following commands in the terminal.
wget -q https://git.io/voEUQ -O /tmp/raspap && bash /tmp/raspap

• Pass Y in every prompt in the terminal.

• Reboot the Device.

• Open the browser in Raspberry Pi and type the following

URL http://localhost or http://raspberrypi.local • Enter Username: “admin” and Password: “secret”

• Navigate to “Hotspot”.

• Click on “Basic”.

• Set the setting as
Interface: wlan0
SSID: tyit (Give name as per your choice)
Wireless Mode: 802.11g-2.4 GHz
Channel: 1 (Select the number devices you want to connect.)

• Press “Save settings”

• Click on “Security”.

• Set the password for your hotspot
PSK: ChangeMe (Anything you want to set.)

• Navigate to System and Press “Reboot”

• Grab your phone and search for your network as you have given in the
SSID.

• Connect.
